# O Experimento

Um jogo de perguntas e respostas desenvolvido em Python com elementos de narrativa interativa.

## Sobre o Projeto

No jogo, o participante é sequestrado por uma organização governamental liderada por **Nefário** e precisa responder corretamente a uma série de perguntas para sobreviver ao experimento.

O jogador enfrenta **10 níveis**, cada um contendo uma pergunta aleatória de conhecimentos gerais, matemática, história, geografia, ciências, inglês e lógica.

## Funcionalidades

* História interativa com diálogos.
* Sistema de pontuação.
* Perguntas selecionadas aleatoriamente.
* 10 níveis de dificuldade.
* Múltiplos finais dependendo do desempenho do jogador.

## Tecnologias Utilizadas

* Python 3
* Biblioteca `random`
* Biblioteca `time`

## Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/o-experimento.git
```

2. Entre na pasta do projeto:

```bash
cd o-experimento
```

3. Execute o arquivo:

```bash
python experimento.py
```

## Sistema de Pontuação

* 0 a 7 acertos: Eliminação.
* 8 acertos: Sobrevive por pouco.
* 9 acertos: Sobrevive.
* 10 acertos: Considerado uma ameaça ao governo.

## Objetivos de Aprendizado

Este projeto foi desenvolvido para praticar:

* Estruturas de repetição (`for`)
* Estruturas condicionais (`if`, `elif`, `else`)
* Listas
* Entrada e saída de dados
* Bibliotecas do Python
* Lógica de programação
